# My Game Website

This is a modern, responsive static website to showcase your game.

## Features
- Homepage with game description
- Features section
- Gallery with placeholder images (replace with your own screenshots)
- Contact form (no backend, for demo purposes)
- Responsive design (works on mobile and desktop)

## How to Use
1. Replace the images in the `images/` folder with your actual game screenshots.
2. Update the text in `index.html` and `contact.html` to match your game's details.
3. Open `index.html` in your browser to view the site.

## Folder Structure
- `index.html` - Main homepage
- `contact.html` - Contact form page
- `styles.css` - Site styling
- `script.js` - JavaScript for interactivity
- `images/` - Placeholder images (replace with your own)
